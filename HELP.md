# ShutterWall Help

Safe first command:
  shutterwall quickstart

Monitoring:
  shutterwall inspect
  shutterwall baseline
  shutterwall watch 3

Preview/apply/undo:
  shutterwall scan
  shutterwall apply
  shutterwall undo

Policy tiers:
  shutterwall scan
  shutterwall scan-business
  shutterwall scan-enterprise

Apply auto-elevation:
  shutterwall apply
  shutterwall apply-business
  shutterwall apply-enterprise

Safety notes:
  - inspect/watch do not apply firewall changes.
  - scan previews only.
  - apply requests Administrator elevation.
  - undo requests Administrator elevation.
  - use only on networks you own or manage.
